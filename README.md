# Djangoshop

## 使用

执行指令：

```bash
pnpm start
```

PNPM 将会自动处理环境杂项并部署 Vue 项目至 Django 中，随后运行 Django 服务。

## 开发

API 接口列表

## 运行

1. 前端

   ```bash
   pnpm dev
   ```

2. 后端

   ```bash
   pnpm serve
   ```
